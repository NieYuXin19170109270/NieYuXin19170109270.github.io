Gitbash基础操作

一、简介

  • 关于Gitbash的基础知识和基本命令行的简单介绍
  • 使用GitBash实现文件/文件夹的创建，删除，复制，转移，查看，解压，重命名等操作

二、命令

  1. cd：切换目录
  2. cd ..：返回上一目录
  3. ls：查看当前目录
  4. ls -a：查看所有文件（包括隐藏文件夹）
  5. ls -R：查看文件夹，以及文件夹中的内容
  6. mkdir：创建文件夹
  7. rm：删除文件
  8. rm -r：删除文件夹
  9. cp：复制文件
  10. cp -r：复制文件夹
  11. cat：查看文件内容
  12. mv：用于移动或重命名文件和目录
  13. vim：创建html文件，并进行并写代码
  14. tar -xf ：解压缩文件

补充：
1、利用*.文件拓展名可以实现一次性对同一类型文件的处理。

三、总结

gitbash是一个在windows系统上运行Git命令的工具，通过使用命令行实现对计算机的操作，更加接近于计算机的底层。

四、补充:如何从本地上传文件到GitHub上？

  1. step1：初始化本地项目

    2. git init

  2. step2：将文件夹中的文件传至git仓库中

    3. git add

  3. step3：将项目提交到git仓库（commit项目）

    4. git commit -m "注释"

  4. step4：新建分支

    5. git branch -M main

  5. step5：将本地仓库（.git）和gitHub上的仓库进行关联

    6. git remote add origin gitHub仓库地址

  6. step6：将本地仓库的项目push到gitHub的远程仓库上

    7. git push -u origin main
